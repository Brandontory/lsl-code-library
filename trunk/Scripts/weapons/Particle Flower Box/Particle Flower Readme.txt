Common commands (owner only):

wave mult #:
(Example: "wave mult 3")
Affects the number used for the internal formula. Try different ones out.

wave dist1 #:
(Example: "wave dist1 6")
Affects outer petal size. Cannot exceed 8.

wave dist2 #:
(Example: "wave dist2 6")
Affects inner petal size. Cannot exceed 8.

color1 <R,G,B>:
(Example: "color1 <0.6,0.8,1>")
Defines the outer petal color in RGB.

color2 <R,G,B>:
(Example: "color2 <0.6,0.8,1>")
Defines the inner petal color in RGB.

color3 <R,G,B>:
(Example: "color3 <0.6,0.8,1>")
Defines the stem color in RGB.
